RPG Character Battle System - COSC 102 Assignment 2

To compile: Navigate to the folder containing the .java files and run:
  javac *.java

To run:
  java Main

No external libraries are required. Developed and tested with Java 11+.
All seven source files (Combatant.java, Describable.java, Abilities.java,
GameCharacter.java, Warrior.java, Mage.java, Rogue.java, Main.java) must
be in the same directory when compiling.
